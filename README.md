# 🌊 Microplastic Pollution Risk Predictor

An advanced environmental intelligence platform that analyzes multiple indicators to predict microplastic contamination risk for any location or Indian city.

## 🔗 Live Website

**[Open the Live Application Here](https://rishitaraj14.github.io/Microplastic-Pollution-Risk-Predictor/)**

## 📋 About

The Microplastic Pollution Risk Predictor helps you understand microplastic exposure risks based on environmental factors. By analyzing key indicators like population density, plastic waste, recycling rates, rainfall patterns, and river proximity, the application provides accurate risk assessments.

### Key Features

- **Manual Predictor** - Customize all parameters to analyze specific locations
- **City Predictor** - Quick risk assessment for Indian cities
- **AI Analysis** - Advanced machine learning-powered predictions
- **Data Visualization** - Interactive charts and graphs for data analysis
- **Dataset Insights** - Explore comprehensive microplastic pollution data
- **Real-time Risk Assessment** - Get instant results based on environmental indicators

## 🌍 Environmental Indicators Analyzed

- 📍 **Population Density** - Higher density often correlates with more plastic waste
- 🗑️ **Plastic Waste** - Direct measure of pollution levels
- ♻️ **Recycling Rates** - Impact of waste management efficiency
- 🌧️ **Rainfall** - Water runoff carrying microplastics
- 🏞️ **River Proximity** - Distance to water bodies affecting contamination

## 🚀 Quick Start

1. Visit the live application: https://rishitaraj14.github.io/Microplastic-Pollution-Risk-Predictor/
2. Select your preferred predictor mode:
   - **Manual Predictor**: Enter custom environmental parameters
   - **City Predictor**: Select an Indian city for instant analysis
   - **AI Analysis**: Get ML-powered predictions
3. View results, charts, and detailed analysis

## 📊 How It Works

The predictor uses a sophisticated algorithm that considers:
- Regional environmental conditions
- Population and urbanization factors
- Waste management practices
- Geographic and hydrological features
- Historical pollution data

Results are categorized as:
- 🟢 **Low Risk** - Safe exposure levels
- 🟡 **Medium Risk** - Moderate concern
- 🔴 **High Risk** - Significant exposure levels
